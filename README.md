# Производственная практика

## Описание
Программа для расчета выполнения сменного задания на нефтехимическом предприятии (АО «Новокуйбышевская нефтехимическая компания»).

### Функциональность
- Ввод названия продукта
- Ввод планового и фактического количества (тонн)
- Расчет процента выполнения
- Вывод результата (выполнен/не выполнен)
- Сохранение результатов в файл shift_results.txt

```bash
python shift_calculation.py
